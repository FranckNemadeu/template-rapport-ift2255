# Glossaire

Ce glossaire définit les termes importants utilisés dans le projet, en particulier ceux qui pourraient prêter à confusion.

| Terme                    | Définition                                                                                                                                 |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **Planifium**            | Système officiel de planification des cours et des programmes de l'Université de Montréal, fournissant les données académiques officielles |
| **Tableau de bord**      | Interface de visualisation agrégée présentant l'ensemble des données et métriques relatives à un cours de manière consolidée               |
| **Étudiant authentifié** | Utilisateur ayant validé son identité via le CAS UdeM et bénéficiant de l'accès complet aux fonctionnalités de la plateforme               |
| **Utilisateur invité**   | Personne accédant à la plateforme sans authentification, avec un accès limité aux fonctionnalités de consultation                          |
| **Avis**                 | Évaluation qualitative et quantitative d'un cours soumise par un étudiant l'ayant suivi, incluant notes et commentaires                    |
| **Charge de travail**    | Estimation du temps hebdomadaire moyen requis pour réussir un cours, exprimée en heures et basée sur les retours d'étudiants               |
| **Difficulté perçue**    | Niveau de complexité subjectif d'un cours tel que rapporté par les étudiants l'ayant suivi, sur une échelle normalisée                     |
| **Sigle de cours**       | Code unique identifiant un cours spécifique selon la nomenclature de l'Université de Montréal (ex: IFT2255)                                |
| **Prérequis**            | Cours ou compétences devant être acquis préalablement à l'inscription à un cours donné                                                     |
| **Coresquis**            | Cours devant être suivis simultanément avec un autre cours                                                                                 |
| **Session académique**   | Période d'enseignement définie (Automne, Hiver, Été) durant laquelle les cours sont dispensés                                              |
| **Modération**           | Processure de validation et de contrôle qualité des avis soumis par les étudiants avant leur publication                                   |
| **Métriques agrégées**   | Statistiques et indicateurs calculés à partir de l'ensemble des données disponibles sur un cours                                           |
| **Profil étudiant**      | Ensemble des caractéristiques académiques et des préférences d'un étudiant utilisé pour la personnalisation                                |
| **Étudiant**             | Personne inscrite à l'Université de Montréal suivant des cours                                                                             |
| **TGDE**                 | Direction des études de premier cycle, responsable des programmes                                                                          |
| **Crédits**              | Unités de valeur académique attribuées à un cours                                                                                          |
| **Moyenne**              | Résultat académique moyen d'un groupe d'étudiants dans un cours                                                                            |
| **Échecs**               | Nombre d'étudiants n'ayant pas réussi un cours                                                                                             |
| **Bot Discord**          | Programme automatisé collectant les avis sur la plateforme Discord                                                                         |
| **Horaire**              | Planning des séances d'un cours pour une session donnée                                                                                    |
| **Profil personnel**     | Préférences et caractéristiques académiques d'un étudiant                                                                                  |
